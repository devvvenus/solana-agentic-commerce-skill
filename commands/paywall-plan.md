# Paywall Plan Command

Use this command text when asking an agent to design a paid Solana API route.

```text
Use $solana-agentic-commerce to plan a Solana stablecoin paywall for this route:

- Framework:
- Method/path:
- Product:
- Price/token:
- Recipient:
- Network:
- Auth requirements:
- Fulfillment behavior:

Return the protocol choice, payment contract, route changes, replay protection, settlement verification, and test matrix.
```
